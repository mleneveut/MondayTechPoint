How to create and deploy your own presentations ?


Prerequisites :
  • node.js : www.nodejs.org

Installation

First, clone the project : 
                         
$ git clone https://github.com/mleneveut/MondayTechPoint.git
In the repository, create your own branch, and switch to it : 
                         
$ git checkout -b "[presentation_name]"
Install dependencies : 
                         
 $ npm install
Create your slides / themes (we'll see it later)
Configure multiplex-http plugin (reveal-config.js + reveal-config-client.js, later again)
Commit and push the modifications on the new branch : 
                         
$ git commit -am "Created my presentation..."
                         
                     
                         
$ git push origin "[presentation_name]"

Creating your own themes


Make sure you have Ruby and Sass installed

Creating your own themes


Copy a theme (/css/theme/source/*.scss)
Override variables (template/*.scss) in your copied file
Add your theme.scss to the Gruntfile (sass task)

Once finished, run : 
                             
 $ grunt themes
Include it in your html
Done !

Installation

Servers :

Master : controls the presentation
Public presentation : application to connect to see the presentation

Prerequisites :
  • node.js
  • express.js


Master application : controls the show

Public presentation application : same project as the master except that it doesn't have secret

Installation

Install node.js : www.nodejs.org

Clone project : 
                           
$ git clone https://github.com/hakimel/reveal.js.git
Install all needed dependencies : 
                           
$ npm install
Notes

Configuration

Add these dependencies : 
                            
Reveal.initialize({
    // Other options

    dependencies: [
        // Other dependencies

        { src: 'plugin/multiplex-http/master.js', async: true },
    ]
});

Master

Launch multiplex-http plugin 


$ node plugin/multiplex-http
Grab a token :
Go to http://yourserver.com/multiplex-http/token
Copy an paste socketId + token into your config file
Specify socket.io server url

The secret value gives master access to the presentation The socketId create unique channel to communicate

Master

Configure multiplex-http : 
                            
multiplexHttp: {
    secret: '13921084506082212166', // Used to grant master access control
    id: '6c33a488a7abd3df', // Obtained from multiplex-http server, create unique channel to communicate
    url: 'http://localhost' // Location of multiplex-http server
},
Design your slides
