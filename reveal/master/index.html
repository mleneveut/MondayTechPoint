How to create and deploy your own presentations ?

First, clone the project : 
                         
                              $ git clone https://github.com/...
In the repository, create your own branch, and switch to it : 
                         
$ git checkout -b "[presentation_name]"
Create your slides / themes (we'll see it later)
Configure multiplex (reveal-config.js + reveal-config-client.js, later again)
Commit and push the modifications on the new branch : 
                         
$ git commit -am "Created my presentation..."
                         
                     
                         
$ git push origin "[presentation_name]"

Deploy your presentation in "production"

Go to :
Type your branch into the form
Deploy !

Creating your own themes


Make sure you have Ruby and Sass installed

Creating your own themes


Copy a theme (/css/theme/source/*.scss)
Override variables (template/*.scss) in your copied file
Add your theme.scss to the Gruntfile (sass task)

Once finished, run : 
                             
                                 $ grunt themes
Include it in your html
Done !

Installation

Servers :

Master (with notes) : controls the presentation and displays notes
Public presentation server : application to connect to see the presentation

Prerequisites :
  • node.js


Master : controls the show

Socket server : receives slide change events and broadcast to client

Public presentation server : same project as the master except that it doesn't have secret

Installation

Install node.js : www.nodejs.org

Clone project : 
                           
                               $ git clone https://github.com/hakimel/reveal.js.git
Install all needed dependencies : 
                           
                               $ npm install
Install node-static for static server purposes : 
                           
                               $ npm install -g node-static
Notes

Configuration

Add these dependencies : 
                            
Reveal.initialize({
    // Other options

    dependencies: [
        // Other dependencies

        { src: '//cdnjs.cloudflare.com/ajax/libs/socket.io/0.9.10/socket.io.min.js', async: true },
        { src: 'plugin/multiplex/master.js', async: true },
        // speaker notes
        { src: 'plugin/notes-server/client.js', async: true },
    ]
});

Master

Launch socket.io : 


                                $ node plugin/multiplex
Get a token :
Go to http://yourserver.com:1948/token
Copy an paste socketId + token into your index.html
Specify socket.io server url

The secret value gives master access to the presentation The socketId create unique channel to communicate

Master

Configure multiplex : 
                            
multiplex: {
    secret: '13921084506082212166', // Used to grant master access control
    id: '6c33a488a7abd3df', // Obtained from socket.io server
    url: '127.0.0.1:1948' // Location of socket.io server
},

Client

                                    secret: null

                                    $ node plugin/notes-server
